# Day 1 iPhone Install

Day 1 is now set up as an installable web app.

To put it on your iPhone with GitHub Pages:

1. Create a new GitHub repository named `Day-1`.
2. Upload the files from this folder, not just the zip file.
3. Open the repo Settings > Pages.
4. Choose `Deploy from a branch`.
5. Select the `main` branch and `/root`, then save.
6. Open the GitHub Pages link in Safari on your iPhone.
7. Tap the Share button.
8. Tap Add to Home Screen.
9. Name it `Day 1`, then tap Add.

Important: the current `file:///Users/devondawson/Documents/Video%20Game/index.html` page can open on your Mac, but iPhone home-screen install needs a real web link. Once hosted, the app icon, standalone full-screen mode, and offline cache can activate.

For a native App Store or TestFlight app, the next step is wrapping this same game with an iOS shell such as Capacitor and building it in Xcode.
