# Day 1

Day 1 is a 16-bit workout RPG that runs in the browser and can be installed on iPhone from Safari once hosted.

## Play Locally

Open `index.html` in a browser.

## Publish With GitHub Pages

1. Create a new GitHub repository named `Day-1`.
2. Upload the files from this folder, not the zip file itself.
3. In GitHub, open Settings > Pages.
4. Set Source to `Deploy from a branch`.
5. Pick the `main` branch and `/root`, then save.
6. Open the Pages link on your iPhone in Safari.
7. Tap Share > Add to Home Screen.

The app includes a web app manifest, iPhone icons, and a service worker for install/offline support once it is served from an HTTPS link.
